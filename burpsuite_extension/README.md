
## Description

### 🚀 Overview
This Pull Request introduces a significant upgrade by migrating the standalone HTLogin Python script into a fully integrated **Burp Suite Extension**. 

By porting the logic to Jython, security researchers can now perform automated login vulnerability assessments directly from the Burp Suite interface without switching to a terminal. This integration allows for better handling of sessions, cookies, and real-time analysis of intercepted traffic.

### ✨ Key Features
* **Native UI Tab:** A new dedicated tab called **"HTLogin Pro"** provides a real-time console to monitor findings and scan progress.
* **Contextual Workflow:** Users can now trigger a scan by simply right-clicking any HTTP request (Proxy, Repeater, etc.) and selecting **"Run HTLogin Scanner"**.
* **Logger & Site Map Integration:** Every request generated by the tool is automatically added to Burp's native **Logger** and **Site Map**, ensuring full transparency and allowing for further manual inspection.
* **Comprehensive Payloads:** Includes all original payloads from the initial project:
    * **SQL Injection** (Classic & Auth bypass)
    * **NoSQL Injection** (JSON-based logic)
    * **XPath Injection**
    * **LDAP Injection**
    * **Default Credentials** (Brute-force logic for common admin pairs)
* **Optimized Performance:** Uses multithreading to ensure the Burp Suite UI remains smooth and responsive during intensive scans.
* **Utility Controls:** Added a "Clear Logs" button for easier management of multiple scanning sessions.

### 🛠 Technical Implementation
The extension is written in **Python 2.7 (Jython)** to be compatible with the Burp Suite Extender API. It leverages:
* `IBurpExtender` for core functionality.
* `ITab` for the custom graphical interface.
* `IContextMenuFactory` for the right-click integration.
* `callbacks.makeHttpRequest` and `callbacks.addToSiteMap` for seamless traffic logging.

### 📖 Installation Instructions
1.  Download the [Jython Standalone JAR](https://www.jython.org/download.html).
2.  In Burp Suite, go to **Extender > Options** and point the **Python Environment** to the Jython JAR.
3.  Go to **Extender > Extensions**, click **Add**, select **Extension Type: Python**, and load the `htlogin_burp.py` file.
